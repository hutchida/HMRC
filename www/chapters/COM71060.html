COM71060

Enquiry Into Return For The Wrong Period

0

This subject is presented as follows.

1

Scope of enquiry where return made for the wrong period

2

Scope of enquiry where return made for the wrong period

3

Where it appears that a company has made a return for the wrong period, the power to enquire into the return extends to the period for which the return should have been made.

4

The scope of the enquiry is similarly extended where a return has become a return for the wrong period as a result of a direction under Section 12(5A) ICTA 1988, as to the accounting date to be used where the company carries on several trades.

5

When is a period the ‘wrong period’?

6

There are two situations where a company may make a return that is a return for the wrong period. The first is where the company makes a return for a period that is treated in the return as an AP, but it is not an AP. The second is where the return is made on the basis that there is no AP ending in or at the end of the specified period, but there is such an AP.

COTAX

15

When you are ready to issue the closure notice to end an enquiry into a return made for the wrong period, you must first amend the company’s AP record to show the correct period or periods. See COM71061 for further information.

16

You then use function RAMA (Record / Amend Assessment) to issue the closure notice from the AP for which the return should have been made.

17

You should include a free format message to make it clear that as part of the enquiry conclusions, the closure notice also designates the period for which the return should have been made. When issued, the closure notice shows the start and end dates of the correct AP.
