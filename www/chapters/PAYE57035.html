PAYE57035:

Multiple Batches

Old version

From a Health and Safety perspective, smaller batches are safer and easier to handle and distribute for capture work for a large scheme. If the logged forms P11D for an employer exceeds the current batch size (set nationally), ECS will split them into multiple batches and display the message ‘Creating Multiple Batches’

  • Select the [OK] button and the message ‘Dispatch Cover Sheet for Batch No. (Number) has been printed’ is displayed

    • A cover sheet will be printed for each batch created when you select the [OK] button on the message

    The batch numbers allocated may not follow on in sequence

  • Select the [OK] button to return to the Workflow Batch Logging screen to start a new batch

    • The Multiple Batches screen only displays a maximum of 50 batch numbers. The recommended batch size is 250 forms P11D, this allows you to log a maximum of 2500 individual forms P11D (not P11D Lists) for an employer. If the maximum is exceeded a message ‘Creating Multiple Batches’ is displayed.

    When you select the [OK] button on the message, ‘Too many workflow batches’ is displayed’

  • Select the [OK] button on this message and log the employer into several batches
