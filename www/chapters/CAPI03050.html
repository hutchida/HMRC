CAPI03050:

Customs Handling Of Import And Export Freight (CHIEF)

Old version

Information available on the CHIEF system can be accessed using the following commands:

DCOMDisplay Commodity Code
  
DTMSDisplay Tariff Measure
DCVCDisplay Countervailing Charges
DNBDDisplay Noticeboard
DMIPDisplay Minimum Import Price
DLICDisplay Licence
DLLUDisplay Licence Usage
DEVDDisplay Entry Details

CHIEF noticeboards give details of important new changes but are not retained on a historical basis.

The relevant six digit code should be input after keying in DNBD.
