From April 2012, you can use EBS Function VIEW EMPLOYER RTI SUBMISSION to look at any Full Payment Submission (FPS) or Employment Payment Summary (EPS) returns that an employer has submitted by the RTI Service.

1

From 6 April 2013 you can use EBS Function VIEW EMPLOYER RTI SUBMISSION to look at any Earlier Year Update (EYU) returns that an employer has submitted or any EYU (internal adjustments) that have been made in RTI Service.

2

There are two ways to retrieve this submission data these are by either

3

Using the data the return was received through the RTI Service Or

4

Using the pay period that the RTI return relates to

5

Where you are trying to request a large amount of data, the following message will be displayed ‘Data collection in progress - viewing not yet available’. This message will also provide you with the option to ‘Cancel’ or ‘Retry’.

6

Once the request has been made the RTI return information will continue to be gathered together regardless of with option you choose.

7

If you decide to select ‘Retry’ you should do this every 5 to 10 minutes until the data is available to view.

8

If you decide to ‘Cancel’ the system will continue to gather the data requested. You can then go back and re-enter the request after waiting 5 to 10 minutes, if the retrieval has been completed the data will then be viewable. If the systems are still gathering the requested data you will be presented with the ‘Data collection in progress - viewing not yet available’ message again.

9

Once a request has been made the RTI submission data gathered will be stored and will be available for instant retrieval for the rest of that working day.

10

11

By using the ‘Received’ option you can view the RTI submissions that were received by the RTI Service in a particular period.

12

13

Selecting received dates between 09/05/2012 and 25/07/2012 will display a list of all the RTI submissions received from and to the dates selected.
