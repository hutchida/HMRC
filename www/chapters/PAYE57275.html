PAYE57275:

Exiting A Batch

Old version

\\lngoxfdatp16vb\Fabrication\Build\0Y02\Data\PAYE57275.xml

0

A batch can be exited without being ‘Completed’ or ‘Suspended’. To do so, select the [Exit] button on the Capture Batch screen.

1

The batch will remain open with the status of ‘In Capture’. The next time you select the [Capture] button on the P11D Management screen, the Capture Batch screen will be populated with the open batch.

2

If the batch has been ‘Completed’ or ‘Suspended’, to return to the P11D Management screen you must select the [Exit] button.
